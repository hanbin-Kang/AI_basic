## 인공지능 수업 복습

실제 암환자 / 인공지능이 암이라고 예측 -> P(암이 기준인데 암이라고 예측했으니깐)/ 암이라고 예측했는데 환자의 상태가 암이니깐 T 

TP  

실제 암환자/ 인공지능이 정상이라고 예측 -> N(암이 기준인데 암이 아니라고 예측했으니깐)/ 정상이라 예측했는데 암이니깐 F

FN   

정상환자/ 인공지능이 암이라고 예측 -> P(암이 기준인데 암이라고 예측)/ 암이라고 예측했는데 암이 아니니깐 F

FP  

정상환자/ 인공지능이 정상이라고 예측 -> N(암이 기준인데 정상이라고 예측)/ 정상이라고 예측했는데 정상이니깐 T

TN  


이 문제에서는 "암"이라는 문제니깐 정상이라고 예측했는데 암인경우가 가장 중요하겠네, 즉 FN이 가장 중요하겠네
그럼 이 문제는 재현율을 정밀도보다 높게 생각해야 하는 문제네 

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

실제 스팸메일/ 인공지능이 스팸이라고 예측(스팸이 기준인데 스팸이라고 예측했으니 P)/ 스팸이라 예측했는데 실제 스팸이니깐 T

TP  

실제 스팸메일/ 인공지능이 일반메일이라고 예측(스팸이 기준인데 일반으로 예측했으니 N)/ 일반으로 예측했는데 스팸이니 F

FN  

실제 일반메일/ 인공지능이 스팸으로 예측(스팸이 기준인데 스팸으로 예측했으니 P)/ 스팸으로 예측했는데 실제로 일반이니깐 F

FP  

실제 일반메일/ 인공지능이 일반으로 예측(스팸이 기준인데 일반으로 했으니 N)/ 일반으로 예측했는데 일반이니 T

TN

이 문제는 "스팸 메일"이 문제니깐 스팸이 아닌 정상 메일인데 스팸으로 판단하는 문제가 가장 중요하겠네, 즉 FP가 가장 중요하겠네
그럼 이 문제는 정밀도를 재현율보다 높게 생각해야 하는 문제네

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

즉 P나 N은 기준에 따라 인공지능이 어떻게 예측을 했냐에 따라 바뀜( 기준이 x인데 인공지능이 x라고 예측하면 P, 기준이 x인데 인공지능이 y라고 판단하면 N)

T나 F는 인공지능의 예측이 맞냐 아니냐에 따라 바뀜(인공지능이 x라고 예측했는데 실제로 x면 T, 인공지능이 x라고 예측했는데 실제는 y면 F)

T나 F는 판단하는거고
P나 N은 기준에 따라 인공지능이 어떻게 예측하는지

🔥 한 줄 핵심 요약
👉 P/N = 모델이 뭐라고 예측했냐
👉 T/F = 그게 맞았냐 틀렸냐

🧠 문제 1 (개념 꼬기형)

어떤 은행에서 대출 사기 탐지 모델을 만들었다.

Positive = "사기 거래"
모델이 어떤 거래를 사기라고 예측했다
실제로는 정상 거래였다

👉 이 경우는 무엇인가?
일단 기준은 "사기"이다
인공지능이 "사기"라고 판단했으니깐 P다
그러나 실제는 "정상" 인공지능의 판단이 틀렸으니 F다
즉 FP이다

🧠 문제 2(조금 더 사고 필요)

한 쇼핑몰에서 VIP 고객 추천 모델을 만든다.

Positive = VIP 고객
VIP 고객에게만 특별 혜택 제공 (비용 발생)

상황:

일반 고객을 VIP로 잘못 분류하면 → 비용 손해  => FP인 상황에서 문제발생
VIP를 놓치면 → 기회 손실 => 인공지능이 FN으로 판단(VIP인데 VIP가 아닌 일반고객이라 판단(N), 사실 VIP고객이니(F))

FN vs FP 둘중 뭐가 더 중요하냐

👉 질문:

1) FP / FN 중 무엇이 더 중요한가? 
=> 비지니스 적인 문제에서는 돈을 버는 것도 중요하지만 돈을 잃는것이 더 중요함. 
즉 FP인 상황에서 문제가 발생함 "VIP"고객으로 인공지능이 판단(P)했는데 사실은 일반 고객(F) 즉 인공지능의 판단이 틀렸을 떄 문제가 발생함

2) Precision / Recall 중 무엇을 더 중요하게 봐야 하는가? 
=>Presicion(정밀도)를 더 중요하게 생각해야함. 예를 들어 스팸메일을 기준으로 인공지능이 스팸메일이라고 판단(P)했는데 사실 일반메일(F)이면 문제가 발생하는 경우와 동일

🧠 문제 3 (진짜 시험 느낌)

어떤 보안 시스템에서:

"위험한 사용자를 절대 놓치면 안 된다" 
=> "위험한 사용자"가 기준이네 
인공지능이 위험한 사용자라 판단하고, 실제로 위험하지 않은 사람 vs 인공지능이 위험하지 않은 사람이라 생각하고, 실제로 위험한 사람
후자가 더 중요하네, 즉 인공지능이 위험하지 않은 사람으로 판단(N), 실제로 위험한사람(F) FN인 경우가 중요함
FP인 경우는 당한 사람은 안타깝지만 어쩔수 없는 경우

👉 질문:

1) FP / FN 중 무엇을 줄여야 하는가? 
=> FN이 중요하기 때문에 FN을 실수하는 경우를 줄여야한다

2) Precision / Recall 중 무엇이 더 중요한가?
=> 예를 들어 암이 기준, 인공지능이 암이 아니라고 판단(N), 사실은 암인 환자(인공지능의 판단이 틀렸음으로 F) FN 이 경우와 동일. 
아닌데 맞다고 하는거는 헤프닝인데, 맞는데 아니라고 하는거면 진짜 위험하기 때문에
즉 Recall(재현율)이 더 중요하다

🧠 문제 4 (역으로 묻기)

어떤 모델의 특징:

Precision(정밀성) = 매우 높음 
Recall(재현율) = 매우 낮음 
=> 정밀성이 재현율보다 높으니 이 경우는 스팸메일과 같은 상황임: FP (아닌데 맞다고 하는게 위험! 맞다고 했는데 아닌경우는 헤프닝)
즉 FP가 거의 없음

👉 이 모델의 성향으로 가장 적절한 설명은?

A. 웬만하면 Positive라고 판단한다
B. 매우 확실할 때만 Positive라고 판단한다
C. FN이 거의 없다
D. FP가 매우 많다

=> B이다
정밀도가 높다는 것은 TP / (TP + FP)인데 FP(사실이라고 예측했는데 사실이 아닌 경우)가 낮아야 1에 가깝게 가기 때문에 정밀도가 높음
재현율이 낮다는 것은 TP / (TP + FN)인데 FN(사실이 아니라고 예측했는데 사실인 경우)이 높아야 0에 가까워 지기 때문에 재현율이 낮음
