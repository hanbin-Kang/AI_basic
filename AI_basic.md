# 인공지능 기초 

인공지능(AI)  
=> 인간의 지능적 행위(학습, 인지, 추론, 언어이해 등의)를 컴퓨터 프로그램으로 모방하는 기술

지능  
=>복잡한 문제를 해결하고 목표 달성을 위해 학습, 이해, 추론하는 능력

--------------------------------------------------------------
# 초기 AI
Rule-based(룰 기반)접근: 
프로그래머가 직접 'if-then'규칙을 작성  
한계: 복잡한 현실 세계에선 예외가 너무 많아 프로그래머의 피로가 누적되고, 모든 예외를 규칙으로 만들지 못하여 정확도가 낮을 가능성이 높다   
즉 새로운 상황(규칙으로 만들지 못한 예외)에 유연하게 대응하기 힘들다  

# 기계학습 (ML)
데이터로부터 패턴 학습  
: 사람이 규칙을 하나하나 작성하지 않고, 데이터로부터 패턴을 학습하여 예측이나 의사결정을 수행하는 기술  
(AI란 지능적 행위를 모방하는 기술이여서 이때부터 진짜 인공지능이라고 할 수 있다)   
학습 과정(1): 데이터 -> 모델(학습)  
학습 과정(2): 새로운 데이터 -> 학습된 모델 -> 학습된 패턴을 바탕으로(일반화) 예측  
ML에서는 Target 설정(모델이 학습을 통해 수행하고자 하는 기능)과 Dataset(그 기능을 학습하기 위해 사용하는 데이터의 집합)이 중요하다

## ML의 대표적인 세가지 학습 유형
- 지도 학습(Supervised Learnig)  
정답(Label)을 이용해 입력(Input)과 출력(Output)의 관계를 학습시키는 방법 
 

ex) 개 vs 고양이  
개의 사진(Data)과 고양이 사진(Data)을 비슷한 비율로 여러장 준비하여 초기 Model에 학습시킨다.  
고양이 사진을 보고 개라고 하면 그 사진은 고양이라고 반복하여 학습시킨다.  
어느정도 완성 됐다 싶을 때 학습된 모델(Trained Model)에 새로운 데이터(학습시킬 때 사용하지 않은 사진)을 넣어 잘 맞추는지 판단한다.

- 비지도 학습(Unsupervised Learnig)
정답(Label)없이 패턴 학습하는 방법
사용 시기: 프로그래머도 정답을 모를 때

ex) 롯데백화점 고객 군집화  
프로그래머는 어떤 기준으로 고객을 VIP 고객인지, 충성 고객인지 , 저활성 고객인지 판단해야하는지 모른다.   
이럴때 사용하는 것이 비지도 학습이다. 인공지능의 모델은 구매 금액, 방문 빈도, 최근 구매 시점 등의 패턴을 바탕으로 고객을 군집화 할 수 있다. 

- 강화 학습  
원하는 결과가 알려져 있지만 실천하기 위한 정확한 경로를 알지 못하는 경우에 사용하는것이 적합하다.  
여러 시행착오를 통해 학습하고 옳은 방향이면 보상(Reward)하고 옳지 않은 방향이면 피드백하면서 원하는 결과로 수렴하도록 한다.

ex) 탁구 로봇  
탁구 로봇의 목표는 공을 맞추고, 상대 코트에 보내는 것이다.  
로봇이 팔을 움직여 탁구채로 공을 맞추면 보상하고 맞추지 못하면 피드백하는 과정을 반복하여 원하는 결과로 수렴하게 한다.


# 딥러닝(DL)
다층 인공신경망(Deep Neural Network)을 사용해 고차원, 비선형 패턴을 학습  
이미지, 음성, 텍스트 등의 비정형 데이터 처리에 강점을 보인다
